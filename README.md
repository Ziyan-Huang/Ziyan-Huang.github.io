# Academic Homepage Template

This is a simple static academic homepage inspired by [Yongtai Zhuo's website](https://yutazhuo.github.io/).

## Files

- `index.html`: page structure and editable content
- `styles.css`: visual design and responsive layout

## How to use

1. Open `index.html` in a browser to preview it locally.
2. Replace the placeholder name, bio, links, news, and publications with your own information.
3. Add your personal photo by replacing the `YN` avatar block if desired.

## Deploy

You can publish this site easily with GitHub Pages by putting these files into a GitHub repository and enabling Pages.
